# Install
## Debian
```
apt install libgmp-dev libmpfr-dev libmpc-dev
pip install hwtypes
```
## OSX
```
brew install gmp mpfr libmpc
pip install hwtypes
```

## CentOS
```
yum install libmpc-devel mpfr-devel gmp-devel
pip install hwtypes
```
